What's this for?

Mostly, it's to provide instructions for the current play testers.

Start: Windows(PC)

  1. Download a vanilla client
  2. Right-click wow-client-1.12.1.zip and select 'Extract All...' to unzip the files.
  3. Open the folder wow-client-1.12.1.
  4. Double-click WoW.exe to launch the game.
  5. Enjoy!

Start: macOS

Run Windows via Boot Camp or VMWare Fusion and follow the instruction for Windows
