What's this for?

Mostly, it's to provide instructions for the current play testers.

Start: Windows (PC)

  1. Suppose you 1have a 2vanilla client
  2. Move wow-client-1.12.1 to the Desktop
  3. Right-click wow-client-1.12.1 and select "Extract All..." to extract the files.
  4. Open the folder wow-client-1.12.1.
  5. Double-click WoW to launch the game.

NOTE: the game may work at the second launch as "Compatibility settings have been applied".

Start: macOS

Run Windows via Boot Camp or VMWare Fusion and follow the instruction for Windows
